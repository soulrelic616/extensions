Features

Improves interaction with Brackets by adding many functionalities.

  • a tab bar similar to the browser
  • drag & drop tabs
  • drag & drop filePath
  • button close file (include middle click)
  • {{{MMB}}}
  • information about file in tooltip
  • a menu with multiple options [move list, close list, ...]
  • includes an icon pack for hundreds of files
  • icons in gray scale mode
  • {{{CWRO}}}
  • button for toggle sidebar
  • move the toolbar to the bottom of the sidebar
  • a button (half circle) to the left half of the editor that slides the sidebar when it is closed
  • add/retrive bookmarks by click on gutter
  • [ContextMenu] copy Path File, remove empty lines

Accesses this configuration through the Navigate menu
